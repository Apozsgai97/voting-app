## Used concepts

* Tech stack: Next.js, TypeScript
* Feature sliced architecture.
* Routing layer separated from the service layer.
* Functional core separated from the imperative shell.

## Installation 

* Install dependencies 
```bash
   npm install
   ```
* Run the server
```bash
   npm run dev
   ```
* Run the tests 
```bash
   npm test
   ```
